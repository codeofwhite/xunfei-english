# 英语学习软件前端

## 目录结构
```plaintext
engTeach/
├─ index.html
├─ package.json
├─ vite.config.js
├─ public/        # 静态资源：书本内容、音频
├─ src/
│  ├─ assets/     # 图片、音频
│  ├─ components/ # Vue 组件
│  └─ views/      # 页面
└─ README.md
```

## 启动步骤
```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 打包发布（上线用）
npm run build
```
